# ec-bigint

Realiza operações aritméticas (soma de dois pontos, duplicação de ponto e potência de ponto) sobre pontos na curvas elíptica $Y^2 + a1XY + a3Y = X^3 + a2X^2 + a4X + a6 
$ módulo $p$. Código implementado em Rust utilizando a crate BigInt.

Referências:
